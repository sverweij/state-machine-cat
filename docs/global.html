playground: state-machine-cat in a global

Open the console and play around there - state-machine-cat is available in window.smcat.

Example of what you could drop in there:

console.log(smcat.version);

let vectorString = smcat.render(
  `initial => blank;
   blank => greeted: greeter says hello;
   greeted => final;`,
  {inputType: 'smcat', outputType: 'svg', direction: 'left-right'}
);
let vectorElement = new DOMParser()
  .parseFromString(vectorString, 'image/svg+xml')
  .documentElement;
document.body.appendChild(vectorElement);
